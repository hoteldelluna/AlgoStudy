#### 너무 어렵지 않은 완전 탐색 문제집 5번문제 !!

<br>

`문제풀이`

- 9개의 입력값을 순열을 이용해 해결

<br>

`code`

```python
import itertools

arr = []
for _ in range(9):
    arr.append(int(input()))

for i in itertools.permutations(arr,7):
    if(sum(i) == 100):
        res = sorted(i)
        break
print(*res, sep="\n")
```
