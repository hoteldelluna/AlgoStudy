`문제풀이`

- 중복있는 순열을 구하면서 오름차순을 만족시키는 결과를 구하는 문제
- 이전 값을 넘겨주는 방식으로 구현

<br>

`code`

```python
def f(n,m,k,l):
    if(n == k):
        print(*res)
        return
    else:
        for i in range(m):
            if(arr[i] >= l):
                visited[i] = 1
                l = arr[i]
                res[n] = arr[i]
                f(n+1,m,k,l)
                visited[i] = 0

n, m = map(int, input().split())
arr = range(1,n+1)
visited = [0 for i in range(n)]
res = [0] * m
f(0,n,m,0)
```
