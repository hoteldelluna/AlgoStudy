## 백준 15651 n과m 3번 -python

<br>

`문제풀이`

- 중복없는 순열을 구하는 문제
- n과m 1번 문제와 기본 로직은 비슷하다
- 같은 값 처리만 뺴주면 된다

<br>

`code`

```python
def f(n,m,k):
    if(n == k):
        print(*res)
        return
    else:
        for i in range(m):
            visited[i] = 1
            res[n] = arr[i]
            f(n+1,m,k)
            visited[i] = 0


n, m = map(int, input().split())
arr = range(1,n+1)
visited = [0 for i in range(n)]
res = [0] * m
f(0,n,m)
```
